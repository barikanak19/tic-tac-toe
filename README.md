# ❌⭕ Tic Tac Toe – React App

A classic Tic Tac Toe game built using React and Vite.

## 🚀 Features
- Two Player Mode
- Winner Detection
- Game Reset
- Score Tracking
- Responsive UI

## 🛠 Tech Stack
- React
- Vite
- CSS
